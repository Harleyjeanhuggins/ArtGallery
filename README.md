## Interactive Art Gallery

An immersive virtual art gallery built with HTML, CSS, and JS. This project features an interactive door that opens when the user
clicks on it, allowing visitors to enter the digital gallery displaying framed artwork.

## Features:
-click to open the door
-10+ framed artwork displays
-smooth CSS animations and transitions
-responsive gallery layout
-custom-designed gallery environment

  
## Technologies used:
-HTML
-CSS
-JavaScript

  
## How it works:
1. The visitor arrives at the gallery entrance.
2. The user interacts with the door by clicking.
3. The door animation opens the gallery.
4. The user can explore the artwork displayed.
   
## Future Improvements:
-Add background music with ambient effects
-Improve mobile responsiveness
-Add additional gallery hallways
-Add/fix light and shadow effects
   
## How to open:
1. Download the project files
2. Open the folder
3. double-click `index.html`
   
## Inspiration:
This project was created to combine web development with digital art presentation.
I wanted my finished artwork to be displayed where you can see them all together.
